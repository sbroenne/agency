Quickstart

Install Bradygaster Squad and adopt a listed squad

Today, Awesome Squads points you to each squad's source repo.

Awesome Squads is for the Bradygaster Squad ecosystem. Install Squad first, then use a listing's source repository and manifest metadata to copy, adapt, or import a squad into your own repo.

1

Install Squad

Install Bradygaster's Squad CLI for GitHub Copilot.

npm install -g @bradygaster/squad-cli
2

Initialize your repo

Create or open your project, then scaffold Squad.

mkdir my-project && cd my-project
git init
squad init
3

Pick a squad

Browse Awesome Squads, open the listing, and follow the source repository link to the canonical implementation.

4

Adapt or import

Copy the squad files into your repo, or use squad import <file> if the source publishes an export snapshot. Then open Copilot and select Squad.
