Quickstart

Install and adopt a squad when you're ready

Helpful, but secondary to browsing the directory.

Show install and adoption steps

Awesome Squads points you to each squad's source repo and manifest metadata. Use this after you've already found a squad worth adopting.

1

Install Squad

Install Bradygaster's Squad CLI for GitHub Copilot.

npm install -g @bradygaster/squad-cli
2

Initialize your repo

Create or open your project, then scaffold Squad.

mkdir my-project && cd my-project
git init
squad init
3

Pick a squad

Browse this directory, inspect the detail panel, and follow the source repository link to the canonical implementation.

4

Adapt or import

Copy the squad files into your repo, or use squad import <file> if the source publishes an export snapshot. Then open Copilot and select Squad.
